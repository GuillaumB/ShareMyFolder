ShareMyFolder
======================================

Description
-------------

ShareMyFolder est un projet personnel qui consiste à réaliser une interface web dans laquelle on pourrait afficher, récupérer et uploader des dossiers et fichiers situé sur notre propre serveur de fichiers.

Pré-requis
------------

- Disposer d'un serveur Web (Apache ou autre)
- Et avoir les droits pour copier le dossier sur votre serveur

Installation
--------------

- Modifier le fichier data/file.php en remplaçant le chemin contenu dans la variable $folderPath (ligne 7) par le chemin absolu du répertoire que vous souhaitez explorer
- Copier l'ensemble du projet sur votre serveur Web
- Rendez-vous à l'adresse du site et commencer l'exploration
